# Segmentify-Challenge
Segmentify Front-end Developer Challenge

## Installation
```
$ git clone git@github.com:SedatBasaran14/segmentify-challenge.git
$ cd segmentify-challenge
$ python -m http.server
$ open http://localhost:8000
```
